# Random Number Generator

## Abstract

A Random Number Generator (RNG) is a fundamental tool in computer science used to generate numbers that lack any predictable pattern. This project implements a random number generator that produces values within a specified range using algorithmic techniques. The application demonstrates how computers simulate randomness through pseudo-random number generation, which is widely used in simulations, cryptography, gaming, statistical sampling, and data analysis.

This project is designed to be simple and easy to understand, making it suitable for beginners learning about randomness and programming concepts.

---

## Features

* Generates random numbers within a user-defined range
* Simple and beginner-friendly implementation
* Fast and efficient random number generation
* Easy to modify and extend for other applications

## Usage

1. Run the program.
2. Enter the desired range for random number generation.
3. The program will output a random number within the specified range.

Example Output:

```
Enter minimum number: 1
Enter maximum number: 100
Random number generated: 57
```
